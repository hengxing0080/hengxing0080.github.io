short s1=1; s1=s1+1有什么错？

short s1=1; s1+=1;有什么错？

第一个：
是有错的，short在内存中占2个字节，而整数1默认为int型占4个字节，
s1+1其实这个时候就向上转型为int类型了，因此第一行代码必须强转才行。

第二个：
之所以可以是以为这句话翻译过来就是s1++，也就是short类型的数据自身加增1，因此不会有问题。
